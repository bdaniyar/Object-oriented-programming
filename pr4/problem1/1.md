1. What is the core difference between a class and an interface?
Class can have implementation (methods with body) and state (fields), while interface defines only a contract (methods without implementation, mostly).

2. Can interfaces have fields? If yes, what are the implicit modifiers for them?
Yes, interfaces can have fields. They are implicitly public, static, and final.

3. Can a class implement multiple interfaces? Explain why this is useful.
Yes, a class can implement multiple interfaces. This is useful because it allows achieving multiple inheritance of behavior and makes code more flexible.
